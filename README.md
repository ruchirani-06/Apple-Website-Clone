# Apple Website Clone

This is a responsive Apple website clone made using HTML, CSS and JavaScript. The design is inspired by Apple's official website.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Fully responsive layout for all devices
- Simple and clean UI like Apple website
- Navigation and basic interactions
- Product sections and layout design

## Files
- index.html
- shop.html
- contact.html
- style.css
- script.js
- img folder

## About This Project
This project was made for practice to improve frontend skills and understanding of responsive design.
